
Docker Registry

A Docker registry stores Docker images and allows you to share them.
-----------------------------------------------------------------------------------------
Types of Registries:

Docker Hub: Public & official images.

Private Registry: Self-hosted for private projects.

Cloud Registries: AWS ECR, Google GCR, Azure ACR.

----------------------------------------------------------------------------------------
Basic Commands:

docker login                          # Login to Docker Hub
docker push <username>/<image>:<tag>  # Push image
docker pull <username>/<image>:<tag>  # Pull image

----------------------------------------------------------------------------------------
