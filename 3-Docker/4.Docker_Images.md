
Docker Images

Docker images are read-only templates containing the application and its dependencies. Images are used to create containers.
----------------------------------------------------------------------------------
Basic Commands:

docker pull ubuntu          # Pull an image
docker images               # List images
docker rmi <image_name>     # Remove an image

-----------------------------------------------------------------------------------

Creating an Image Using Dockerfile:

# Dockerfile example
FROM ubuntu
RUN apt-get update && apt-get install -y python3
CMD ["python3", "--version"]

docker build -t my-python-image .
